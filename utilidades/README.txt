EASYTAS
Descripción
	EASYTAS es una aplicación web diseñada para ayudar a las personas a encontrar profesionales de diferentes áreas (psicólogos, 		expertos en nutrición, desarrolladores de software, entre otros) de manera rápida y sencilla.
Tecnologías utilizadas
	HTML5
	CSS3
	Google Fonts
	Visual Studio Code con extensiones:
	Live Server (para ejecutar el proyecto en el navegador en tiempo real)
	Prettier (para formatear el código)
Estructura del proyecto
	El proyecto está organizado de la siguiente forma (última actualización: 3 de septiembre de 2025):
	css/ → Contiene los estilos del proyecto.
	img/ → Contiene las imágenes utilizadas.
	javascript/ → Carpeta destinada a futuros scripts JS.
	php/ → Carpeta destinada a la integración backend en PHP.
	LandPageEasyTas.html → Archivo principal de la aplicación (landing page).
	EASYTAS LANDING PAGE.pdf → Documento con la propuesta visual del proyecto.
Instalación y uso
	Clonar o descargar el repositorio en su equipo.
	Abrir el proyecto en Visual Studio Code.
	Instalar y habilitar la extensión Live Server.
	Ejecutar el archivo LandPageEasyTas.html con Live Server para visualizar la aplicación en el navegador.
